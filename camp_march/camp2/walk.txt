Прогулка


Поиск пункта назначения в большом незнакомом городе может быть сложной задачей, особенно если вы пытаетесь найти кратчайший путь. С учетом карты города вы хотите
найти кратчайший путь между текущим положением и пунктом назначения.
Карта города представляется на плоскости в виде бесконечной сетки состоящей из единичных квадратов.
В настоящее время вы находитесь на площади (0, 0) и пункт назначения - квадрат (X, Y).
Есть N зданий в городе. Каждое здание представляет собой прямоугольник полностью занимающих несколько единичных квадратов. Никакие два здания не пересекаются и не касаются. Здания определяется заданием
координат двух диагонально-противоположных квадратов занятых зданием.
На каждом шаге, вы можете ходить в одному из четырех соседних квадратов, но не можете ходить на квадраты
занимаемые зданиями. Ваше текущее положение - на западном въезде в город и х-координата каждого
квадрата, занимаемого зданием строго больше нуля.
Напишите программу, которая, учитывая расположение зданий, находит кратчайшие пути от вашего текущего
положения к месту назначения. Путь следует представлять в виде последовательности вертикальных и горизонтальных сегментов, ни какие 
два последовательных сегмента не должны быть параллельны. Длина пути - число квадратов, содержащихся в пути,
за исключением начального квадрата.


Первая строка содержит два целых числа X, Y (1 ≤ X ≤ 10^6, -10^6 ≤ Y ≤ 10^6) - координаты
пункта назначения. Вторая строка содержит одно целое число N (0 ≤ N ≤ 100 000) - число
зданий в городе. Каждая из следующих строк N содержит четыре целых числа X1, Y1, X2, Y2 (1 ≤ X1, X2 ≤ 10^6,
-10^6 ≤ Y1, Y2 ≤ 10^6) - координаты двух противоположных по диагонали квадратов занятых зданием.


Первая строка вывода должна содержать целое число L - длину кратчайшего пути к месту назначения.
Вторая строка должна содержать целое число M - число сегментов в кратчайшие пути.
Количество сегментов M не должна превышать 1.000.000.
Каждая из следующих M строк должна содержать два целых числа, DX и DY, описывающих движение в 
одном сегменте. Для каждого сегмента, ровно одно из значений DX, DY должно быть равно нулю.
Примечание: если есть несколько решений, вы должны вывести любое из них.


input

9 1
2
5 -3 8 3
10 -3 13 3

output

16
3
0 4
9 0
0 -3


input

12 0
5
2 -1 3 1
6 -7 8 -1
6 1 8 6
4 3 4 5
10 -5 10 3

output

24
8
0 -2
4 0
