Очередь


Пара билетов осталась на финальный матч и огромная очередь образовалась перед билетной кассой. Болельщики были помечены последовательностью целых чисел по мере прибытия. Первый человек в очереди был помечен номер один, второй - номером два и т.д.
Так как болельщики прибыли накануне игры, они должны были ждать в течение долгого времени, прежде чем касса открылась, и, естественно, некоторые из них были вынуждены использовать туалет. Каждый раз, когда человеку нужно в туалет, он выходит
из очереди и, после завершения своих дел, возвращается в очередь, не обязательно в то же
место, как раньше. Поскольку есть только один туалет, человек не уходит из очереди пока предыдущей
человек не вернулся (и, следовательно, в любой момент не более одного человека отсутствует в очереди).
В течение ночи, в общей сложности было N посещений туалета. Каждый визит описывается двумя положительными
целыми числами А и В, обозначая, что человек помеченный A вышел из очереди, а затем сделал вернулся в
очередь непосредственно перед человеком помеченным В. Теперь, когда все посещения завершились нужно
ответить на ряд вопросов. Каждый вопрос - это либо 'Р X' (на какой позиции стоит человек X), или 'L X' (какой человек стоит на позиции X)
Первый человек в очереди считается в позиции один, второй в позиции два и т. д.
Напишите программу, которая, при заданном описании посещений и количестве вопросов, ответит на все
вопросы.


Первая строка содержит целое число N (2 ≤ N ≤ 50 000) - общее число посещений уборной.
Каждая из следующих строк N содержит два различных целых числа A и B (1 ≤ A, B ≤ 10^9), описывающий один туалет
визита. Следующая строка содержит целое число Q (1 ≤ Q ≤ 50 000) - общее количество вопросов.
Каждая из следующих строк содержит один символ Q (заглавная буква 'Р' или заглавная буква
'L') и целое число X (1 ≤ X ≤ 10^9), описывающие один вопрос.


Вывод должен состоять из Q строк.
i-я строка вывода должна содержать одно целое число R - ответ на i-й вопрос.
