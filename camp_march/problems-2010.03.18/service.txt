Mobile Service

Компания предоставляет услуги для своих партнеров, которые расположены в разных городах. У компании
имеется три мобильных сотрудника. Если происходит запрос в каком-то месте, сотрудник
должен перейти от его текущего местоположения на место запроса (если там нет 
сотрудника). Только один сотрудник может двигаться в любой
момент. Они могут двигаться только по запросу и не могут быть в одном и том же месте.
Перемещение сотрудника с места p на место q стоит C(p, q). Функция
стоимости не обязательно симметричная, но С (р, р) = 0.
Компания должна удовлетворять запросы по правилу первый пришел, первым обслужен. Цель состоит в том,
чтобы свести к минимуму общую стоимость обслуживания данной последовательности запросов.

Вы должны написать программу, которая решает, какой работник обслуживающего персонала должен совершить переход для
каждого запроса, так, чтобы общая стоимость обслуживания данной последовательности была минимально возможной.


Первая строка service.in содержит два целых числа, L и Н. L (3 <= L <= 200) - количество мест и N (1 <= N <= 1000) - количество запросов. Места
определяются целыми числами от 1 до L. В каждой из следующих L строк L содержится L неотрицательных
целых чисел. j-е число в строке i+1 - стоимость C(i, j), и не превышает 2000. Последняя строка
содержит N целых чисел - список запросов. Запрос определяется идентификатором
места, где он происходит. Первоначально, сотрудники находятся в местах 1, 2 и 3, соответственно.

Первая строка service.out содержит одно целое число M - минимальную общую стоимость
обслуживания входной последовательности запросов (подзадача A). Вторая строка содержит ровно N
целых чисел. i-е число - идентификатор сотрудника (1, 2 или 3), который будет
обслуживать i-й запрос (подзадача B). Если есть несколько вариантов ответа, выведите любой. Вы можете пропустить вторую строку, если Вы
решаете только подзадачу А.

service.in

5 9
0 1 1 1 1
1 0 2 3 2
1 1 0 4 1
2 1 5 0 1
4 2 3 4 0
4 2 4 1 5 4 3 2 1

service.out

5
1 2 1 2 2 1 3 1 3
