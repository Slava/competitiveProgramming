Ссылки


Асхат реорганизацует сайт своей школы. На сайте много траниц и 
содержание хорошее, но он заметил, что страницы не правильно связаны между собой. В самом деле, каждая страница содержит ровно одну 
ссылки, указывающие на другую страницу на сайте. Это плохой дизайн - начиная от домашней страницы, посетитель , как правило, должен пройти по нескольким ссылки, для достижения нужной страницы, а некоторые страницы могут быть недостижимы с главной страницы. В качестве первого шага, он хочет, чтобы добавить несколько ссылок, так что каждая страница могла быть быстро 
доступна с главной страницы. Новые ссылки можно добавить в любом месте сайта. 
Сайт содержит N страниц с пометкой в виде целого числа от 1 до N (домашняя страница отмечена номером 1). 
Есть также N ссылок, каждая страница содержит ровно одну ссылку, указывающую на некоторую другую страницу. Для целого 
К веб-сайт называется K-достижимым, если каждая страница на сайте, кроме домашней страницы может быть достигнута 
с главной страницы, следуя максимум по K ссылкам. 
Напишите программу, которая, для заданного описания сайта и целого K, находит минимальное число 
ссылок, которые должны быть добавлены для того, чтобы сделать сайт K-достижимым.

Первая строка содержит два целых числа N и K (2 ≤ N ≤ 500 000, 1 ≤ K ≤ 20 000).
Каждая из следующих N строк содержит два различных целых числа A и B (1 ≤ A, B ≤ N) что означает, что ссылка на странице A указывает на страницу B.

Первая и единственная строка вывода должна содержать одно целое число - минимальное количество дополнительных ссылок, требуемых, чтобы сделать сайт K-достижима.


input
8 3
1 2
2 3
3 5
4 5
5 6
6 7
7 8
8 5

output
2
