Дан неориентированный граф без петлей и параллельных ребер. Для каждых двух различных вершин существует
ровно один общий сосед (вершина не является соседом сама себе). Нужно обрабатывать запросы на удаление ребер и
нахождение кратчайшего расстояния между двумя вершинами.

Входные данные: n, m (3 <= n <= 100000). Затем m строк по два числа - описание дорог.
Затем до конца файла запросы:
DELETE x - удалить дорогу с индексом x (нумеруются с 1 в том порядке, в котором даны во входном файле)
LENGTH x y - вывести расстояние между городами x и y.
Общее количество запросов не превышает 200000.

input

3 3
1 2
2 3
3 1
LENGTH 1 2
DELETE 1
LENGTH 1 2
LENGTH 2 3
DELETE 3
LENGTH 1 2

output

1
2
1
-1
