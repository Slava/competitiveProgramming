yЗадача 10.

Описание
Подсчитайте площадь заданного многоугольника.

Формат входных данных
В первой строке входного файла находится число N (3 <= N <= 50 000) – количество вершин многоугольника. Последующие N строк содержат по 2 целых числа x и y (−10 000 <= x, y <= 10 000) – координаты вершин

Формат выходных данных
Выведите в выходной файл площадь многоугольника с точностью 600 знаков после запятой.

Примеры:
ввод	вывод
3       0.500000000000000...000
0 0
1 0
0 1
Выходной файл содержит 600 нулей.

Задача 11.

Описание
Даны вещественные числа a, b, c, d. Выяснить, можно ли в прямоугольник со сторонами a, b целиком поместить прямоугольник со сторонами c, d.

В первой строке входного файла находятся вещественные числа a, b, c и d, разделенные одним или несколькими пробелами или символами перевода строки.

Выведите в выходной файл слово YES, если второй прямоугольник можно поместить в первый, или слово NO в противном случае.

Примеры:
ввод	        вывод
317 10 11 23    NO
31 10 10 31     YES

yЗадача 12.

Описание
Задано две линии уравнениями: a1X+b1Y+c1 = 0 a2X+b2Y+c2 = 0 Требуется найти точку их пересечения.

Формат входных данных
В первой строке входного файла записаны коэффициенты первого уравнения a1,b1,c1, во второй коэффициенты второго a2,b2,c2. Известно, что коэффициенты не равняются нулю, и линии не идентичны.

Формат выходных данных
В выходной файл вывести координаты точки пересечения линий с точностью до двух знаков.

Примеры:
ввод	вывод
3 4 5   1.00 -2.00
1 2 3
